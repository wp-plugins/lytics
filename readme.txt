This plugin integrates Lytics into your WordPress site. It includes support for common use management tools like Restrict Content Pro as well as common form builders including Gravityforms and Ninjaforms.

== Description ==

*Lytics is a data hub that lets you create and share segments across all marketing tools.*

You will need to sign up for a Lytics account at [www.getlytics.com](http://www.getlytics.com)

This plugin currently supports a limited number of plugin integrations. It is easy to add integrations, and more will be available soon. If you don't see one, just ask. We might add it to a future version.

*   "WordPress Core Support" addon sends email addresses to Lytics any time someone fills out the default comment form or log into your WordPress site.
*   "Restrict Content Pro" addon sends the *subscription* data for registered subscribers, if they are logged in.
*   "Jetpack" addon sends the email address entered in the *subscribe* widget, if activated.
*   "Gravityforms" addon enables support for the Gravityforms plugin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `lytics` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Sign up for a Lytics account and locate the "script tag" provided in the account activation instructions.
4. Configure the Lytics plugin to use the script tag.
5. (optional) Enable desired addons.

== Frequently Asked Questions ==

= Does this replace Google Analytics? =

No.

= Does your plugin work with my theme? =

This plugin works with any theme provided that it uses the build in comment form.
